# Power User Extensions

## Available Commands

Run these slash commands in Claude Code:

- `/cook <task>` - Full orchestration: research, plan, implement, test, review
- `/bootstrap` - Initialize project with best practices
- `/scout` - Find files across codebase
- `/plan <feature>` - Create implementation plan
- `/fix <issue>` - Debug and fix issues
- `/design <component>` - UI/UX design workflow
- `/code <task>` - Focused coding session
- `/docs <action>` - Documentation management
- `/git <action>` - Git workflow helpers
- `/test <scope>` - Run and analyze tests

## Subagents Available

Use Task tool with these subagent types:
- `code-reviewer` - Code quality analysis
- `debugger` - Issue investigation
- `tester` - Test execution
- `planner` - Implementation planning
- `researcher` - Research and discovery
- `docs-manager` - Documentation
- `ui-ux-designer` - Design workflows
- `database-admin` - Database operations
- `project-manager` - Project tracking
- `git-manager` - Git operations
